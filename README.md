# Mathemizer

A crate for calculating basic mathematical expressions.

# Usage

```rust
use mathemizer::math::{
    add,
    divide,
    multiply,
    subtract,
};

fn main() {
    println!("Hello, world!");
    println!("{}", add(1, 2));
    println!("{}", subtract(1, 2));
    println!("{}", multiply(1, 2));
    println!("{}", divide(1, 2));
}

```
